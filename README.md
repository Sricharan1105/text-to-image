# 🎨 WordWeave – AI Text-to-Image Generator

WordWeave is an AI-powered Text-to-Image Generator that transforms user prompts into stunning AI-generated images using the Hugging Face FLUX.1-dev model.

Built with HTML, CSS, and JavaScript, the project provides a clean and responsive interface for generating creative visuals from text descriptions.

---

## 🚀 Features

- ✨ Generate images from text prompts
- 🤖 AI-powered image generation
- 🎨 Modern responsive UI
- ⚡ Fast and interactive experience
- 📱 Mobile-friendly design
- 🖼️ Download generated images
- 🌈 Custom background support

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Hugging Face API
- FLUX.1-dev Model

## ⚙️ How It Works

1. Enter a text prompt
2. Prompt is sent to the Hugging Face API
3. AI generates an image
4. Generated image is displayed on screen
5. User can download the image

## 💻 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/WordWeave.git
```

### Open project folder

```bash
cd WordWeave
```

### Run the project

Open `index.html` in your browser.

## 🔑 API Configuration

Add your Hugging Face API key inside `script.js`

```javascript
const API_URL = "YOUR_API_URL";
const API_KEY = "YOUR_API_KEY";
```
